# DistribuMax S.A. — Sistema de Gestión de Inventario y Logística

Prototipo operacional desarrollado en Java con persistencia en MySQL.
Materia: Seminario de Informática
Trabajo Práctico: AP2 — Segundo Seminario

## Tecnologías utilizadas

- Lenguaje: Java (JDK 17)
- Base de datos: MySQL 8.0
- Conectividad: JDBC (Java Database Connectivity)
- Patrón: MVC (Modelo-Vista-Controlador)

## Entregables del TP2

- Modelo de análisis (diagrama de secuencia)
- Modelo de diseño (diagrama de clases)
- Modelo de implementación (diagrama de despliegue)
- Modelo de pruebas (plan y casos de prueba)
- Diagrama Entidad-Relación (DER)
- Base de datos MySQL: creación, inserción, consulta y borrado
- Definiciones de comunicación
- Prototipos de interfaz Java Swing

## Contenido del repositorio

- `distribumax_db.sql` — Script SQL completo del sistema
